# Wedding Invitation

Statička web pozivnica spremna za GitHub Pages.

## Lokalno otvaranje

Otvorite `index.html` u browseru ili pokrenite mali server:

```bash
python3 -m http.server 8080
```

## GitHub Pages

Repo može da se objavi preko Settings → Pages → Deploy from branch → `main` → `/root`.

## Podešavanja

Glavni podaci su u `script.js`, a tekstovi u `index.html`.

RSVP trenutno čuva odgovor u browseru i nudi slanje na `djordje.novcic95@gmail.com`. Sledeći korak je povezivanje na Google Sheets, Formspree ili Supabase.
